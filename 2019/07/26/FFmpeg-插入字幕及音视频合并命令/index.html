FFmpeg 插入字幕及音视频合并命令

发表于 2019-07-26 更新于 2019-08-09

插入字幕

分为两种方式

硬字幕

即将字幕与视频合并，并且不能再分割出来

-strict -2 表示使用ffmpeg 自带的 aac编码器
force_style=’Fontsize=30,FontName=FZYBKSJW–GB1-0’
如果不能被识别，可以将\ 去掉试试

软字幕

即内挂字幕，字幕还可以分离出来，插入速度快，但是需要播放器支持

音视频合成

无音轨合并，将一段音频与无音轨的视频进行合并，速度快
带音轨合并多个音频，可指定时间戳，速度慢
